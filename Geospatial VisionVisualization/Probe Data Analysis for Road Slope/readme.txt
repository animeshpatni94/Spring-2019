Language Used:
Java

Instructions: 
1. Copy/Download the Source Files:
	a. Partition6467LinkData.csv
	b. Partition6467ProbePoints.csv
	Into the "data" folder present in the directory where you will extract the zip file.

2. Open the Terminal Window:
	Type:
	For Compiling the code: javac MapMatchingScript.java
	For Running the code: Java MapMatchingScript
	
The code takes a lot of time to run on our systems as the Source Data is huge.
To get output we have used complete Partition6467LinkData.csv and 50,000 points from Partition6467ProbePoints.csv
